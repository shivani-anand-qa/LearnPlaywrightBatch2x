# VS Code Keyboard Shortcuts - Windows

| Action | Shortcut |
|---|---|
| **Single line comment** | `Ctrl + /` |
| **Block comment** | `Shift + Alt + A` |
| **Open terminal** | `Ctrl + `` ` `` |
| **Save file** | `Ctrl + S` |
| **Save all files** | `Ctrl + K S` |
| **Undo** | `Ctrl + Z` |
| **Redo** | `Ctrl + Y` |
| **Copy line down** | `Shift + Alt + ↓` |
| **Copy line up** | `Shift + Alt + ↑` |
| **Move line down** | `Alt + ↓` |
| **Move line up** | `Alt + ↑` |
| **Delete line** | `Ctrl + Shift + K` |
| **Select entire line** | `Ctrl + L` |
| **Multi-cursor** | `Alt + Click` |
| **Select next occurrence** | `Ctrl + D` |
| **Format document** | `Shift + Alt + F` |
| **Go to line** | `Ctrl + G` |
| **Quick open file** | `Ctrl + P` |
| **Command palette** | `Ctrl + Shift + P` |
| **Split editor** | `Ctrl + \` |
| **Toggle sidebar** | `Ctrl + B` |
| **Zoom in** | `Ctrl + =` |
| **Zoom out** | `Ctrl + -` |
| **Find** | `Ctrl + F` |
| **Find & Replace** | `Ctrl + H` |
